/instructions to use this code
Install all necessary dependencies by using the following command line below:
npm i init -y express ejs save-dev nodemon dotenv bcrypt passport passport-local express-session express-flash mongoose

Then, create .env file & type: SESSION_SECRET=secret

To run type: npm run devStart

and then go to http://localhost:3000/
